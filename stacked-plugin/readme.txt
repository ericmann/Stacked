== Description ==

Custom plugin to provide custom post type support for stacked pages and team member profiles for the Genesis child theme, Stacked.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire `/stacked-plugin` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Download and activate a Stacked-based theme.

== Frequently Asked Questions ==

= How do I add a stack? =

Create a new Stack using the WordPress admin sidebar.  The content of the stack is a comma-delimited list of page slugs for the pages you wish to use.
